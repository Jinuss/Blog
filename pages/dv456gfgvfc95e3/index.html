    # lorem 乱数假文

    # 概述

    lorem，即拉丁语 Lorem ipsum，是拉丁语的缩写，是印刷和排版领域所常用的虚拟文字。由于曾经一台匿名的打印机刻意打乱了一盒印刷字体从而造出一本字体样品书，Lorem ipsum 从而传播到印刷业。

    lorem 也称乱数假文或者亚元文本，它的目的就是为了保持字母多多少少标准及样式一致，避免在排版时出现版式问题。
